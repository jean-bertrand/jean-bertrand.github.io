A Healthcare Analytics & Consulting Firm

Ideation and conceptualization for a user interface to inspect and analyze vast amounts of incoming data from firm clients.

Goal was to come up with an interface that is easily digestible for internal and external users in understanding the status of data loads. This interface allows analysts to better communicate with their clients the status of not only each file in the set but also provides detailed look at which fields are missing, mislabeled etc.
